# Quiz43

Quiz43 is a quiz web app built with **Next.js (React)**.  
This repository contains the project source code (see the `quiz43-next/` folder) and an additional `backup/` folder with older/extra files.

> Repo: https://github.com/petronilo43/quiz43

## Tech Stack

- Next.js (React)
- JavaScript
- Node.js / npm

## Getting Started (Local Setup)

### Prerequisites
- Node.js (recommended: **Node 18+**)
- npm (or yarn/pnpm)

### Run the app locally

```bash
# 1) Go to the Next.js project folder
cd quiz43-next

# 2) Install dependencies
npm install

# 3) Start the development server
npm run dev

npm run dev      # Run in development mode
npm run build    # Build for production
npm run start    # Start production server (after build)
npm run lint     # Lint (if configured)
